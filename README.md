# voice-bro-assistant

![assbro](https://github.com/user-attachments/assets/4d1c24aa-4943-4b60-a855-6781c88fdf17)

Голосовой ассистент на Python с полной поддержкой русского языка.
Офлайн-распознавание (Vosk), синтез речи (pyttsx3) и перевод (Argos Translate).

## Возможности

- **Wake word** — активация по слову «бро»
- **Прерывание** — можно перебить ассистента голосом (WebRTC VAD), сразу переходит в режим прослушивания
- **Погода** — текущая погода через OpenWeatherMap (кэш 10 мин, понятные ошибки при отсутствии ключа)
- **Новости** — топ-5 через NewsAPI (кэш 30 мин, понятные ошибки при отсутствии ключа)
- **Заметки** — создание, просмотр, удаление (сохраняются в файл)
- **Напоминания** — простые и с таймингом («через 10 минут», «завтра в 9 утра»)
- **Поиск в интернете** — через DuckDuckGo
- **Музыка** — поиск и стриминг с YouTube (требуется ffmpeg)
- **Перевод** — офлайн-перевод ru↔en (Argos Translate)
- **Калькулятор** — безопасное вычисление выражений (без eval)
- **Скорость речи** — управление голосом (быстрее/медленнее/нормальная/число)
- **Голос** — выбор мужского или женского голоса
- **Шутки** — случайный анекдот
- **Время** — текущее время
- **Логирование** — `app.log` + консоль
- **Стартовая проверка** — при запуске проверяет Vosk-модель, ffplay, API-ключи
- **Персистентность** — скорость речи и голос сохраняются в `settings.json`

## Установка

```bash
pip install -r requirements.txt
```

### Vosk-модель

Скачать [vosk-model-small-ru-0.22.zip](https://alphacephei.com/vosk/models/vosk-model-small-ru-0.22.zip)
и распаковать в корень проекта.

### ffmpeg (для музыки)

Установить ffmpeg (нужен `ffplay` в PATH):
- **Windows:** `winget install ffmpeg` или скачать с [ffmpeg.org](https://ffmpeg.org/)
- **Linux:** `sudo apt install ffmpeg`
- **macOS:** `brew install ffmpeg`

### .env

Создать `.env` в корне проекта (ключи необязательны — без них погода/новости не работают):

```
OPENWEATHERMAP_KEY=your_key
NEWSAPI_KEY=your_key
```

## Запуск

```bash
python main.py
```

Скажите «бро» для активации, затем команду.

| Категория | Фразы |
|---|---|
| Приветствие | привет, добрый день, здравствуйте |
| Прощание | пока, до свидания |
| Погода | погода, какая погода |
| Новости | новости, что нового |
| Время | время, который час |
| Напоминание | напомни [через 10 минут] ... |
| Заметка | заметка ..., запиши ... |
| Показать заметки | покажи заметки, мои заметки |
| Калькулятор | посчитай 2+3*4 |
| Перевод | переведи hello (на русский) / переведи на английский привет |
| Поиск | найди python |
| Музыка | включи Imagine Dragons |
| Стоп музыка | выключи музыку, останови музыку |
| Шутка | шутка, анекдот |
| Скорость | говори быстрее, говори медленнее, нормальная скорость, скорость речи 200 |
| Голос | голос мужской, голос женский |
| Как дела | как дела, как поживаешь |
| Помощь | помощь, что умеешь |
